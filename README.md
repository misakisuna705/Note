## Coding Style

### 命名

#### 內容

| global 完整 |   local 簡潔    |
| :---------: | :-------------: |
|    class    | member function |
|   struct    | member constant |
|  function   | member variable |
|    enum     |                 |
|    macro    |                 |
|  constant   |                 |
|  variable   |                 |

#### 大小寫

|  全大寫  | 首字母大寫 |  全小寫  |
| :------: | :--------: | :------: |
|   enum   |   class    | variable |
|  macro   |   struct   |          |
| constant |            |          |

#### 常用命名

|      |             |
| :--: | :---------: |
| 變數 | value / val |
| 資料 | data / dat  |
| 指標 |     ptr     |
| 陣列 |     arr     |
| 字串 |     str     |
| 數量 | num / size  |
| 迴圈 |  i / j / k  |
